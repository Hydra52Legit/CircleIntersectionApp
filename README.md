# Circle Intersection App

[![.NET](https://img.shields.io/badge/.NET-8.0-blue.svg)](https://dotnet.microsoft.com/)
[![Avalonia](https://img.shields.io/badge/Avalonia-11.0-purple.svg)](https://avaloniaui.net/)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

Курсовая работа: Разработка кросс-платформенного приложения для определения пересечения окружностей с использованием фреймворка Avalonia UI.

## 📋 Описание

Приложение позволяет:
- Вводить координаты и радиусы двух окружностей
- Автоматически вычислять точки пересечения
- Визуализировать окружности и точки пересечения на интерактивном графике
- Загружать данные из файла и сохранять результаты
- Использовать темную тему с адаптивным интерфейсом

## 🚀 Возможности

### Математические вычисления
- ✅ Определение пересечения двух окружностей
- ✅ Вычисление точек пересечения (0, 1 или 2 точки)
- ✅ Обработка всех случаев: пересечение, касание, отсутствие пересечения
- ✅ Поддержка дробных координат и радиусов

### Пользовательский интерфейс
- ✅ Темная эргономичная тема
- ✅ Адаптивный интерфейс (работает на любом размере окна)
- ✅ Интерактивная навигация по графику (зум, панорамирование)
- ✅ Автоматическое позиционирование графика
- ✅ Анимации и плавные переходы

### Технические возможности
- ✅ Кросс-платформенность (Windows, macOS, Linux)
- ✅ Загрузка/сохранение данных в файлы
- ✅ Обработка ошибок и валидация данных
- ✅ Высокая производительность

## 📦 Установка и запуск

### Системные требования
- .NET 8.0 Runtime
- Windows 7+, macOS 10.12+, Linux (Ubuntu 18.04+)
- 256 MB RAM минимум

### Установка

1. **Установите .NET 8.0 Runtime:**
   - Скачайте с [официального сайта](https://dotnet.microsoft.com/download/dotnet/8.0)
   - Или через пакетный менеджер вашей ОС

2. **Склонируйте репозиторий:**
   ```bash
   git clone https://github.com/your-username/CircleIntersectionApp.git
   cd CircleIntersectionApp
   ```

3. **Соберите и запустите:**
   ```bash
   dotnet build
   dotnet run
   ```

## 📖 Использование

1. **Ввод данных:**
   - Введите координаты и радиусы двух окружностей в формате:
   ```
   x₁ y₁ r₁
   x₂ y₂ r₂
   ```
   - Пример: `0 0 5` и `10 0 5`

2. **Навигация по графику:**
   - **Зум:** Колесико мыши или кнопки "+" и "-"
   - **Панорамирование:** Зажмите левую кнопку мыши и двигайте
   - **Сброс вида:** Кнопка "🎯 Сброс вида"

3. **Работа с файлами:**
   - **Загрузить:** Кнопка "📂 Загрузить" для чтения данных из .txt файла
   - **Сохранить:** Кнопка "💾 Сохранить" для экспорта результатов
