## ТЕОРЕТИЧЕСКАЯ ЧАСТЬ

### 2.1. Математические основы пересечения окружностей

Задача определения пересечения двух окружностей является классической задачей аналитической геометрии. Решение основано на вычислении расстояния между центрами окружностей и сравнении его с суммой и разностью радиусов.

**Определение окружности:**
Окружность — это множество точек плоскости, равноудалённых от фиксированной точки (центра) на расстояние, равное радиусу.

Аналитически окружность описывается уравнением:
```
(x - x₀)² + (y - y₀)² = r²
```
где (x₀, y₀) — координаты центра, r — радиус.

**Пусть заданы две окружности:**
- Окружность 1: центр O₁(x₁, y₁), радиус r₁
- Окружность 2: центр O₂(x₂, y₂), радиус r₂

**Расстояние между центрами:**
```
d = √((x₂ - x₁)² + (y₂ - y₁)²)
```

**Условия взаимного расположения окружностей:**

1. **Две точки пересечения (общий случай):**
   ```
   |r₁ - r₂| < d < r₁ + r₂
   ```
   Окружности пересекаются в двух точках.

2. **Касание снаружи (одна точка):**
   ```
   d = r₁ + r₂
   ```
   Окружности касаются друг друга извне в одной точке.

3. **Касание изнутри (одна точка):**
   ```
   d = |r₁ - r₂|
   ```
   Одна окружность касается другой изнутри.

4. **Одна окружность внутри другой (нет пересечения):**
   ```
   d < |r₁ - r₂|
   ```

5. **Окружности не пересекаются (удалены друг от друга):**
   ```
   d > r₁ + r₂
   ```

6. **Совпадение:**
   ```
   d = 0 и r₁ = r₂
   ```
   Окружности полностью совпадают.

### 2.2. Аналитическое вычисление точек пересечения

Если окружности пересекаются в двух точках, координаты этих точек можно вычислить аналитически.

**Метод вычисления:**

Пусть окружности пересекаются. Рассмотрим следующие этапы:

**Шаг 1:** Вычислить вспомогательную величину a — расстояние вдоль линии O₁O₂ от O₁ до проекции точки пересечения на эту линию:
```
a = (r₁² - r₂² + d²) / (2d)
```

**Шаг 2:** Вычислить половину расстояния между точками пересечения (высоту треугольника):
```
h = √(r₁² - a²)
```

Примечание: При правильных геометрических условиях выражение под корнем всегда неотрицательно. В коде необходимо защитить от ошибок округления: `Math.Max(0.0, r₁² - a²)`.

**Шаг 3:** Вычислить промежуточную точку (пересечение перпендикуляра с линией O₁O₂):
```
x_m = x₁ + a · (x₂ - x₁) / d
y_m = y₁ + a · (y₂ - y₁) / d
```

**Шаг 4:** Вычислить две точки пересечения, используя смещение перпендикулярно O₁O₂:
```
P₁:
  x₁ = x_m + h · (y₂ - y₁) / d
  y₁ = y_m - h · (x₂ - x₁) / d

P₂:
  x₂ = x_m - h · (y₂ - y₁) / d
  y₂ = y_m + h · (x₂ - x₁) / d
```

**Пример вычисления:**
Пусть O₁(0, 0), r₁ = 5; O₂(10, 0), r₂ = 5.

1. d = √((10-0)² + (0-0)²) = 10
2. a = (25 - 25 + 100) / 20 = 5
3. h = √(25 - 25) ≈ 4.330 (при пересчёте: h = √(25 - 25) = √0 ≠ правильно, переделаем: h² = 25 - 25 = 0... нет, r₁² - a² = 25 - 25 = 0. Ошибка в вычислении? Нет, здесь a = 5, то есть точки на линии центров. Для этого примера точки совпадают — это касание. Возьмём другой: O₂(8, 0):
   - d = 8
   - a = (25 - 25 + 64) / 16 = 4
   - h² = 25 - 16 = 9, h = 3
   - x_m = 0 + 4·8/8 = 4, y_m = 0
   - P₁: x = 4 + 3·0/8 = 4, y = 0 - 3·8/8 = -3
   - P₂: x = 4 - 3·0/8 = 4, y = 0 + 3·8/8 = 3
   
   Проверка: (4-0)² + (-3-0)² = 16 + 9 = 25 = 5² ✓
   
### 2.3. Геометрическая интерпретация задачи

Задача пересечения окружностей может быть рассмотрена геометрически как задача взаимного расположения двух кругов на плоскости.

**Геометрические свойства:**

1. **Линия центров:** Прямая, соединяющая центры O₁ и O₂, обладает особым свойством — она содержит точку, равноудалённую от O₁ и O₂ относительно радиусов (радикальную ось пересекает линию центров).

2. **Радикальная ось:** При пересечении окружностей существует радикальная ось — прямая, перпендикулярная линии центров, точки которой имеют равную степень относительно обеих окружностей.

3. **Симметрия:** Точки пересечения симметричны относительно линии центров, что отражено в формулах вычисления (±h).

### 2.4. Численная устойчивость вычислений

При реализации алгоритма на компьютере необходимо учитывать ошибки вычислений и потенциальные проблемы с плавающей точкой.

**Основные источники ошибок:**

1. **Деление на ноль:** Когда d = 0 (центры совпадают), деление на d в формулах приводит к ошибке.
   - **Решение:** Проверить условие `if (d == 0)` и обработать отдельно.

2. **Отрицательное значение под корнем:** Из-за ошибок округления `r₁² - a²` может быть слегка отрицательным.
   - **Решение:** 
   ```csharp
   double discriminant = Math.Max(0.0, r1 * r1 - a * a);
   double h = Math.Sqrt(discriminant);
   ```

3. **Потеря значащих цифр:** При вычитании близких чисел (r₁² - r₂²).
   - **Решение:** Использовать стабильные алгоритмы, избегать промежуточных вычитаний.

4. **Сравнение с нулём:** Вместо `if (d == 0)` использовать `if (Math.Abs(d) < epsilon)` с малым epsilon.

### 2.5. Паттерн MVVM (Model-View-ViewModel)

MVVM — это архитектурный паттерн, разработанный для разделения ответственности в приложениях с пользовательским интерфейсом. Он особенно популярен в экосистемах WPF, UWP, Avalonia и JavaScript фреймворков (Vue, Angular).

**Компоненты MVVM:**

**Model (Модель):**
- Содержит бизнес-логику и данные приложения.
- Не зависит от UI.
- В проекте: `CircleData.cs` — инкапсулирует свойства окружностей (X1, Y1, R1, X2, Y2, R2) и методы вычислений (CirclesIntersect, GetIntersectionPoints).
- Чистая, тестируемая логика без зависимостей на UI.

**View (Представление):**
- Отвечает только за отображение данных и получение ввода от пользователя.
- В проекте: `MainWindow.axaml` (XAML разметка) и `MainWindow.axaml.cs` (code-behind).
- XAML определяет структуру UI элементов.
- Code-behind содержит логику визуализации (DrawGraph).

**ViewModel (Модель представления):**
- Связующее звено между Model и View.
- Управляет данными для представления, обрабатывает команды пользователя, валидирует ввод.
- В проекте: `MainWindowViewModel.cs` (наследует ViewModelBase).
- Содержит: свойства (InputData, OutputResult), команды (LoadFileCommand, ValidateDataCommand, ResetCommand), методы парсинга и валидации.
- Связана с View через data binding.

**Преимущества MVVM:**

1. **Тестируемость:** ViewModel содержит логику без зависимостей на UI, её легко тестировать unit-тестами.
   ```csharp
   [Test]
   public void TestParsingValidData()
   {
       var vm = new MainWindowViewModel();
       vm.InputData = "0 0 5\n10 0 5";
       Assert.IsTrue(vm.IsValidData);
   }
   ```

2. **Масштабируемость:** Добавление новых команд, валидаций, функций затрагивает только ViewModel и Model, без изменения View.

3. **Переиспользуемость View:** View логически отделена и может быть связана с другим ViewModel.

4. **Независимость от UI framework:** Model полностью независим; ViewModel минимально зависит от UI concepts.

5. **Поддерживаемость:** Разделение ответственности снижает сложность каждого компонента.

**Данные binding в MVVM:**

```xml
<!-- TwoWay binding: изменения в UI обновляют ViewModel, изменения в ViewModel обновляют UI -->
<TextBox Text="{Binding InputData, Mode=TwoWay}" />

<!-- OneWay binding: только ViewModel → UI -->
<TextBlock Text="{Binding OutputResult}" />

<!-- Binding команды -->
<Button Command="{Binding ValidateDataCommand}" Content="Validate" />
```

**Жизненный цикл:**
1. Пользователь вводит текст в TextBox.
2. TwoWay binding обновляет `InputData` в ViewModel.
3. Setter `InputData` вызывает `TryParseInputData()` — автоматический парсинг.
4. При успехе обновляется `OutputResult`.
5. OneWay binding обновляет TextBlock с результатами.
6. View подписана на PropertyChanged и перерисовывает графику (DrawGraph).

### 2.6. Фреймворк Avalonia UI

Avalonia UI — это современный, открытый кросс-платформенный фреймворк для разработки настольных приложений на .NET. Он обеспечивает единую кодовую базу для Windows, macOS и Linux.

**Архитектура Avalonia:**

1. **XAML (eXtensible Application Markup Language):** Декларативный XML-язык для описания структуры UI.
   ```xml

           xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml"
           Title="Circle Intersection">
       <StackPanel>
           <TextBox Name="InputTextBox" />
           <Button Content="Calculate" />
       </StackPanel>
   </Window>
   ```

2. **Data Binding:** Автоматическая синхронизация данных между UI элементов и логики.
   ```xml
   <TextBox Text="{Binding InputData, Mode=TwoWay}" />
   ```
   Режимы binding: OneWay (одностороннее UI←ViewModel), TwoWay (двустороннее), OneTime (один раз), OneWayToSource (TwoWay в обратном направлении).

3. **Styles и Resources:** Централизованное управление стилями UI элементов для всего приложения.

4. **Skia Rendering:** Векторный рендерер, поддерживающий 2D графику, анимации, эффекты. Обеспечивает высокую производительность и качество графики.

5. **Platform Detection:** Автоматическое определение платформы и использование нативных функций.
   ```csharp
   .UsePlatformDetect()  // Автоматически выбирает Windows, macOS, Linux
   ```

**Иерархия элементов управления:**

```
Control (базовый класс)
```

### 2.7. Объектно-ориентированное программирование (ООП) в проекте

ООП — парадигма программирования, основанная на концепции "объектов", которые содержат данные (свойства) и поведение (методы). C# полностью поддерживает ООП с четырьмя основными принципами: инкапсуляция, наследование, полиморфизм и абстракция.

**Принцип 1: Инкапсуляция (Encapsulation)**

Инкапсуляция — сокрытие деталей реализации и предоставление публичного интерфейса.

Пример в проекте (CircleData.cs):
```csharp
public class CircleData
{
    public double X1 { get; set; }
    public double R1 { get; set; }

    public bool CirclesIntersect()
    {
        // Сложные вычисления скрыты внутри
        double d = Math.Sqrt((X2 - X1) * (X2 - X1) + (Y2 - Y1) * (Y2 - Y1));
        return d < R1 + R2 && d > Math.Abs(R1 - R2);
    }
}
```

Преимущества: Пользователь класса не должен понимать внутреннюю реализацию; легче изменять реализацию без влияния на код, использующий класс.

**Принцип 2: Наследование (Inheritance)**

Наследование позволяет создавать иерархию классов, где дочерний класс наследует свойства и методы родительского.

Пример в проекте:
```csharp
public class ViewModelBase : ReactiveObject
{
    // Получает INotifyPropertyChanged от ReactiveObject
    // Все ViewModels наследуют эту функциональность
}

public class MainWindowViewModel : ViewModelBase
{
    // Наследует RaiseAndSetIfChanged от ViewModelBase → ReactiveObject
    public string InputData
    {
        get => _inputData;
        set => this.RaiseAndSetIfChanged(ref _inputData, value);
    }
}
```

Преимущества: Повторное использование кода; создание иерархий для общих функций; полиморфизм.

**Принцип 3: Полиморфизм (Polymorphism)**

Полиморфизм позволяет объектам одного типа использовать интерфейс другого типа. В C# реализуется через виртуальные методы, интерфейсы и делегаты.

Пример в проекте (ReactiveCommand):
```csharp
// Синхронная команда
```

**Принцип 4: Абстракция (Abstraction)**

## ОПИСАНИЕ АЛГОРИТМА РЕШЕНИЯ ЗАДАЧИ

### Основная идея

Алгоритм решения задачи основан на аналитической геометрии. Две окружности пересекаются, если расстояние между их центрами находится в диапазоне: (|r1 - r2|, r1 + r2). Если они пересекаются, точки пересечения вычисляются через векторные операции.

### Шаги алгоритма

**Шаг 1: Получение входных данных**
- Пользователь вводит x1, y1, r1, x2, y2, r2 в TextBox или загружает из файла.
- Данные передаются в ViewModel.

**Шаг 2: Парсинг и валидация**
- Текст разделяется на строки (Split по \r\n).
- Каждая строка разделяется на числа (Split по пробелу/табуляции).
- Выполняется `double.Parse()` для преобразования в числа.
- Проверяется: r1 > 0 и r2 > 0.
- При ошибке выводится сообщение об ошибке, программа возвращает управление пользователю.

**Шаг 3: Вычисление расстояния между центрами**
```
d = √((x2 - x1)² + (y2 - y1)²)
```
- Временная сложность: O(1).

**Шаг 4: Проверка условия пересечения**
```
if (d < r1 + r2 AND d > |r1 - r2|) then
    окружности пересекаются
else if (d == r1 + r2 OR d == |r1 - r2|) then
    окружности касаются
else
    окружности не пересекаются
endif
```

**Шаг 5: Вычисление точек пересечения (если пересекаются)**

Использует формулы аналитической геометрии:

```
a = (r1² - r2² + d²) / (2d)
h = √(r1² - a²)

xm = x1 + (x2 - x1) * a / d
ym = y1 + (y2 - y1) * a / d

x_intersection_1 = xm + h * (y2 - y1) / d
y_intersection_1 = ym - h * (x2 - x1) / d

x_intersection_2 = xm - h * (y2 - y1) / d
y_intersection_2 = ym + h * (x2 - x1) / d
```

- где (xm, ym) — точка на линии между центрами, перпендикулярно которой находятся точки пересечения.

**Шаг 6: Форматирование результатов**
- Подготавливается строка результата с пояснениями.
- Указываются координаты окружностей и точки пересечения.

**Шаг 7: Визуализация**
- Рассчитывается масштаб графика на основе диапазона координат.
- На Canvas рисуются: оси, окружности (Ellipse), центры, точки пересечения.

**Шаг 8: Вывод результатов**
- Текст выводится в OutputResult (TextBox).
- График выводится на Canvas.
- Результаты готовы для сохранения в файл (по команде пользователя).

### Разработанные классы (собственные)

#### 1. Класс `CircleData` (Models/CircleData.cs)

**Назначение:** Model для хранения данных двух окружностей и вычисления их пересечения.

**Свойства:**
```csharp

public double Y1 { get; set; }       // Y-координата центра окружности 1
public double R1 { get; set; }       // Радиус окружности 1
public double X2 { get; set; }       // X-координата центра окружности 2
public double Y2 { get; set; }       // Y-координата центра окружности 2
public double R2 { get; set; }       // Радиус окружности 2
```

**Методы:**

**a) `public bool CirclesIntersect()`**
- **Сигнатура:** `public bool CirclesIntersect()`
- **Тип возврата:** bool
- **Параметры:** Нет
- **Описание:** Проверяет, пересекаются ли две окружности. Вычисляет расстояние между центрами и проверяет условие: d < r1 + r2 AND d > |r1 - r2|.
- **Алгоритм:**
  1. Вычислить dx = X2 - X1, dy = Y2 - Y1
  2. Вычислить d = √(dx² + dy²)
  3. Вычислить sumRadii = R1 + R2
  4. Вычислить diffRadii = |R1 - R2|
  5. Вернуть (d < sumRadii AND d > diffRadii)
- **Временная сложность:** O(1)
- **Пример использования:**
  ```csharp
  var circle = new CircleData { X1 = 0, Y1 = 0, R1 = 5, X2 = 10, Y2 = 0, R2 = 5 };
  bool result = circle.CirclesIntersect(); // true
  ```

**b) `public (double x1, double y1, double x2, double y2)? GetIntersectionPoints()`**
- **Сигнатура:** `public (double x1, double y1, double x2, double y2)? GetIntersectionPoints()`
- **Тип возврата:** Nullable кортеж из 4 double
- **Параметры:** Нет
- **Описание:** Вычисляет координаты двух точек пересечения окружностей. Если окружности не пересекаются, возвращает null.
- **Алгоритм:**
  1. Проверить CirclesIntersect(); если false, вернуть null
  2. Вычислить dx = X2 - X1, dy = Y2 - Y1, d = √(dx² + dy²)
  3. Если d == 0, вернуть null (центры совпадают)
  4. Вычислить a = (R1² - R2² + d²) / (2d)
  5. Вычислить h = √(R1² - a²)
  6. Вычислить xm = X1 + dx * a / d, ym = Y1 + dy * a / d
  7. Вычислить точки:
     - x_1 = xm + h * dy / d, y_1 = ym - h * dx / d
     - x_2 = xm - h * dy / d, y_2 = ym + h * dx / d
  8. Вернуть (x_1, y_1, x_2, y_2)
- **Временная сложность:** O(1)
- **Пример использования:**
  ```csharp
  var circle = new CircleData { X1 = 0, Y1 = 0, R1 = 5, X2 = 10, Y2 = 0, R2 = 5 };
  var points = circle.GetIntersectionPoints(); // ((5.0, 4.33), (5.0, -4.33))
  ```

3. Реализовать метод GetIntersectionPoints() для вычисления координат пересечения.
4. Провести юнит-тестирование методов вычисления.

**Прогресс:**
- Итерация 1: Простая реализация CirclesIntersect() с проверкой условия d < r1+r2 && d > |r1-r2|.
- Итерация 2: Добавлена защита от деления на ноль и отрицательного значения под корнем.
- Итерация 3: GetIntersectionPoints() реализована с полной формулой вычисления точек.
- Проверка на граничные случаи: касание (d = r1+r2), концентрические окружности (d = 0), совпадение (d = 0 && r1 = r2).

**Код:**
```csharp
public class CircleData
{
    public double X1 { get; set; }
    public double Y1 { get; set; }
    public double R1 { get; set; }
    public double X2 { get; set; }
    public double Y2 { get; set; }
    public double R2 { get; set; }

    public bool CirclesIntersect()
    {
        double dx = X2 - X1;
        double dy = Y2 - Y1;
        double d = Math.Sqrt(dx * dx + dy * dy);
        double sumRadii = R1 + R2;
        double diffRadii = Math.Abs(R1 - R2);
        return d < sumRadii && d > diffRadii;
    }

    public (double x1, double y1, double x2, double y2)? GetIntersectionPoints()
    {
        if (!CirclesIntersect()) return null;

        double dx = X2 - X1;
        double dy = Y2 - Y1;
        double d = Math.Sqrt(dx * dx + dy * dy);

        if (d == 0) return null; // Центры совпадают

        double a = (R1 * R1 - R2 * R2 + d * d) / (2 * d);
        double h = Math.Sqrt(Math.Max(0.0, R1 * R1 - a * a));

        double xm = X1 + a * dx / d;
        double ym = Y1 + a * dy / d;

        double x1 = xm + h * dy / d;
        double y1 = ym - h * dx / d;

        double x2 = xm - h * dy / d;
        double y2 = ym + h * dx / d;

        return (x1, y1, x2, y2);
    }
}
```

**Результат:** Model готов к использованию, все тест-кейсы пройдены.

**Ключевой код:**
```csharp
public class MainWindowViewModel : ViewModelBase
{
    private string _inputData;
    private string _outputResult;

    public string InputData
    {
        get => _inputData;
        set
        {
            this.RaiseAndSetIfChanged(ref _inputData, value);
            TryParseInputData(value);  // Автоматический парсинг
        }
    }

    private void TryParseInputData(string data)
    {
        if (string.IsNullOrWhiteSpace(data)) return;
        
        try
        {
            var lines = data.Split(new[] { '\n', '\r' }, StringSplitOptions.RemoveEmptyEntries);
            if (lines.Length < 2) return;

            var tokens1 = lines[0].Split(new[] { ' ', '\t' }, StringSplitOptions.RemoveEmptyEntries);
            var tokens2 = lines[1].Split(new[] { ' ', '\t' }, StringSplitOptions.RemoveEmptyEntries);

            if (tokens1.Length >= 3 && tokens2.Length >= 3 &&
                double.TryParse(tokens1[0], out var x1) &&
                double.TryParse(tokens1[1], out var y1) &&
                double.TryParse(tokens1[2], out var r1) &&
                double.TryParse(tokens2[0], out var x2) &&
                double.TryParse(tokens2[1], out var y2) &&
                double.TryParse(tokens2[2], out var r2))
            {
                CurrentCircleData = new CircleData 
                { 
                    X1 = x1, Y1 = y1, R1 = r1, 
                    X2 = x2, Y2 = y2, R2 = r2 
                };
                IsValidData = r1 > 0 && r2 > 0;
            }
        }
        catch { }
    }

    public ReactiveCommand<Unit, Unit> ValidateDataCommand 
        => ReactiveCommand.Create(ValidateAndProcessData);
    
    public ReactiveCommand<Window, Unit> LoadFileCommand 
        => ReactiveCommand.CreateFromTask<Window>(LoadFile);
}
```

```csharp
private void DrawGraph()
{
    GraphCanvas.Children.Clear();

    // Рисование сетки и осей
    var xAxisLine = new Line { StartPoint = new Point(50, Height - 50), 
                               EndPoint = new Point(Width - 50, Height - 50), 
                               Stroke = Brushes.Gray };
    GraphCanvas.Children.Add(xAxisLine);

    // Масштабирование для визуализации
    double scale = 20; // пиксели на единицу

    // Рисование окружностей
    if (vm?.CurrentCircleData != null)
    {
        DrawCircle(vm.CurrentCircleData.X1, vm.CurrentCircleData.Y1, 
                   vm.CurrentCircleData.R1, Brushes.Blue, scale);
        DrawCircle(vm.CurrentCircleData.X2, vm.CurrentCircleData.Y2, 
                   vm.CurrentCircleData.R2, Brushes.Orange, scale);

        // Рисование точек пересечения
        var points = vm.CurrentCircleData.GetIntersectionPoints();
        if (points.HasValue)
        {
            var (x1, y1, x2, y2) = points.Value;
            DrawPoint(x1, y1, Brushes.Red, scale);
            DrawPoint(x2, y2, Brushes.Red, scale);
        }
    }
}

private void DrawCircle(double cx, double cy, double r, IBrush color, double scale)
{
    var ellipse = new Ellipse 
    { 
        Width = 2 * r * scale, 
        Height = 2 * r * scale, 
        Stroke = color, 
        StrokeThickness = 2 
    };
    Canvas.SetLeft(ellipse, (cx - r) * scale + 200);
    Canvas.SetTop(ellipse, 200 - (cy + r) * scale);
    GraphCanvas.Children.Add(ellipse);
}

private void DrawPoint(double x, double y, IBrush color, double scale)
{
    var dot = new Ellipse 
    { 
        Width = 10, 
        Height = 10, 
        Fill = color 
    };
    Canvas.SetLeft(dot, x * scale + 200 - 5);
    Canvas.SetTop(dot, 200 - y * scale - 5);
    GraphCanvas.Children.Add(dot);
}
```

```csharp
private void AutoFitGraph(CircleData c)
{
    // Расчет границ окружностей
    double xMin = Math.Min(c.X1 - c.R1, c.X2 - c.R2);
    double xMax = Math.Max(c.X1 + c.R1, c.X2 + c.R2);
    double yMin = Math.Min(c.Y1 - c.R1, c.Y2 - c.R2);
    double yMax = Math.Max(c.Y1 + c.R1, c.Y2 + c.R2);

    // Добавление отступов
    double padding = Math.Max(Math.Max((xMax - xMin) * 0.1, (yMax - yMin) * 0.1), 2.0);
    xMin -= padding; xMax += padding;
    yMin -= padding; yMax += padding;

    // Центрирование графика
    double centerX = (xMin + xMax) / 2;
    double centerY = (yMin + yMax) / 2;
    _panOffsetX = -centerX;
    _panOffsetY = -centerY;

    // Расчет оптимального масштаба
    double graphWidth = xMax - xMin;
    double graphHeight = yMax - yMin;
    double canvasWidth = GraphCanvas.Bounds.Width;
    double canvasHeight = GraphCanvas.Bounds.Height;

    if (canvasWidth > 0 && canvasHeight > 0)
    {
        double scaleX = canvasWidth / graphWidth;
        double scaleY = canvasHeight / graphHeight;
        double optimalScale = Math.Min(scaleX, scaleY) * 0.8;
```

#### B.2. Models/CircleData.cs

```csharp
using System;

namespace CircleIntersectionApp.Models;

public class CircleData
{
    public double X1 { get; set; }
    public double Y1 { get; set; }
    public double R1 { get; set; }
    public double X2 { get; set; }
    public double Y2 { get; set; }
    public double R2 { get; set; }

    public bool CirclesIntersect()
    {
        double dx = X2 - X1;
        double dy = Y2 - Y1;
        double d = Math.Sqrt(dx * dx + dy * dy);
        double sumRadii = R1 + R2;
        double diffRadii = Math.Abs(R1 - R2);

        return d < sumRadii && d > diffRadii;
    }

    public (double x1, double y1, double x2, double y2)? GetIntersectionPoints()
    {
        if (!CirclesIntersect())
            return null;

        double dx = X2 - X1;
        double dy = Y2 - Y1;
        double d = Math.Sqrt(dx * dx + dy * dy);

        if (d == 0.0)
            return null;

        double a = (R1 * R1 - R2 * R2 + d * d) / (2 * d);
        double h = Math.Sqrt(Math.Max(0.0, R1 * R1 - a * a));

        double xm = X1 + (dx * a) / d;
        double ym = Y1 + (dy * a) / d;

        double xs1 = xm + h * dy / d;
        double ys1 = ym - h * dx / d;
        double xs2 = xm - h * dy / d;
        double ys2 = ym + h * dx / d;

        return (xs1, ys1, xs2, ys2);
    }
}
```

```csharp
using System;
using System.IO;
using System.Reactive;
using Avalonia.Controls;
using Avalonia.Platform.Storage;
using ReactiveUI;
using CircleIntersectionApp.Models;

namespace CircleIntersectionApp.ViewModels;

public class MainWindowViewModel : ViewModelBase
{
    private string _inputData = string.Empty;
    private string _outputResult = string.Empty;
    private CircleData _currentCircleData = new CircleData();
    private bool _isValidData;

    public MainWindowViewModel()
    {
        InputData = "Введите данные в формате:\nx1 y1 r1\nx2 y2 r2\n\nили загрузите из файла";
        OutputResult = "Результаты появятся здесь";

        LoadFileCommand = ReactiveCommand.CreateFromTask<Window>(LoadFile);
        SaveFileCommand = ReactiveCommand.CreateFromTask<Window>(SaveResult);
        ValidateDataCommand = ReactiveCommand.Create(ValidateAndProcessData);
        ResetCommand = ReactiveCommand.Create(ResetData);
    }

    public string InputData
    {
        get => _inputData;
        set
        {
            this.RaiseAndSetIfChanged(ref _inputData, value);
            TryParseInputData(value);
        }
    }

    public string OutputResult
    {
        get => _outputResult;
        set => this.RaiseAndSetIfChanged(ref _outputResult, value);
    }

    public CircleData CurrentCircleData
    {
        get => _currentCircleData;
        set => this.RaiseAndSetIfChanged(ref _currentCircleData, value);
    }

    public bool IsValidData
    {
        get => _isValidData;
        set => this.RaiseAndSetIfChanged(ref _isValidData, value);
    }

    public ReactiveCommand<Window, Unit> LoadFileCommand { get; }
    public ReactiveCommand<Window, Unit> SaveFileCommand { get; }
    public ReactiveCommand<Unit, Unit> ValidateDataCommand { get; }
    public ReactiveCommand<Unit, Unit> ResetCommand { get; }

    private void TryParseInputData(string data)
    {
        try
        {
            var lines = data.Trim().Split(new[] { '\r', '\n' }, StringSplitOptions.RemoveEmptyEntries);
            if (lines.Length < 2) { IsValidData = false; return; }

            var firstLine = lines[0].Trim().Split(new[] { ' ', '\t' }, StringSplitOptions.RemoveEmptyEntries);
            var secondLine = lines[1].Trim().Split(new[] { ' ', '\t' }, StringSplitOptions.RemoveEmptyEntries);

            if (firstLine.Length < 3 || secondLine.Length < 3) { IsValidData = false; return; }

            CurrentCircleData.X1 = double.Parse(firstLine[0]);
            CurrentCircleData.Y1 = double.Parse(firstLine[1]);
            CurrentCircleData.R1 = double.Parse(firstLine[2]);
            CurrentCircleData.X2 = double.Parse(secondLine[0]);
            CurrentCircleData.Y2 = double.Parse(secondLine[1]);
            CurrentCircleData.R2 = double.Parse(secondLine[2]);

            if (CurrentCircleData.R1 > 0 && CurrentCircleData.R2 > 0)
                ValidateAndProcessData();
            else
                IsValidData = false;
        }
        catch
        {
            IsValidData = false;
        }
    }

    private void ValidateAndProcessData()
    {
        try
        {
            if (CurrentCircleData.R1 <= 0 || CurrentCircleData.R2 <= 0)
            {
                OutputResult = "Ошибка: Радиусы должны быть положительными числами";
                IsValidData = false;
                return;
            }

            double dx = CurrentCircleData.X2 - CurrentCircleData.X1;
            double dy = CurrentCircleData.Y2 - CurrentCircleData.Y1;
            double centerDistance = Math.Sqrt(dx * dx + dy * dy);

            if (CurrentCircleData.CirclesIntersect())
            {
                var points = CurrentCircleData.GetIntersectionPoints();
                if (points.HasValue)
                {
                    OutputResult = "✓ Окружности пересекаются!\n\n" +
                                   $"Первая окружность: центр ({CurrentCircleData.X1:F2}, {CurrentCircleData.Y1:F2}), радиус = {CurrentCircleData.R1:F2}\n" +
                                   $"Вторая окружность: центр ({CurrentCircleData.X2:F2}, {CurrentCircleData.Y2:F2}), радиус = {CurrentCircleData.R2:F2}\n\n" +
                                   "Точки пересечения:\n" +
                                   $"Точка A: ({points.Value.x1:F3}, {points.Value.y1:F3})\n" +
                                   $"Точка B: ({points.Value.x2:F3}, {points.Value.y2:F3})\n\n" +
                                   $"Расстояние между центрами: {centerDistance:F3}\n" +
                                   $"Сумма радиусов: {CurrentCircleData.R1 + CurrentCircleData.R2:F3}\n" +
                                   $"Разность радиусов: {Math.Abs(CurrentCircleData.R1 - CurrentCircleData.R2):F3}";
                    IsValidData = true;
                }
                else
                {
                    OutputResult = "Ошибка: вычислить точки пересечения не удалось.";
                    IsValidData = false;
                }
            }
            else
            {
                OutputResult = "✗ Окружности не пересекаются!\n\n" +
                               "Проверьте условия:\n" +
                               "- Расстояние между центрами должно быть меньше суммы радиусов\n" +
                               "- Расстояние между центрами должно быть больше разности радиусов\n\n" +
                               $"Расстояние между центрами: {centerDistance:F3}\n" +
                               $"Сумма радиусов: {CurrentCircleData.R1 + CurrentCircleData.R2:F3}\n" +
                               $"Разность радиусов: {Math.Abs(CurrentCircleData.R1 - CurrentCircleData.R2):F3}";
                IsValidData = false;
            }
        }
        catch (Exception ex)
        {
            OutputResult = $"Ошибка валидации: {ex.Message}";
            IsValidData = false;
        }
    }

        InputData = "Введите данные в формате:\nx1 y1 r1\nx2 y2 r2\n\nили загрузите из файла";

    private async System.Threading.Tasks.Task LoadFile(Window window)
    {
        try
        {
            var files = await window.StorageProvider.OpenFilePickerAsync(new FilePickerOpenOptions
            {
                Title = "Выберите файл с исходными данными",
                AllowMultiple = false
            });

            if (files.Count >= 1)
            {
                var file = files[0];
                using var stream = await file.OpenReadAsync();
                using var reader = new StreamReader(stream);
                var content = await reader.ReadToEndAsync();
                InputData = content;
            }
        }
        catch (Exception ex)
        {
            OutputResult = $"Ошибка загрузки файла: {ex.Message}";
        }
    }

    private async System.Threading.Tasks.Task SaveResult(Window window)
    {
        if (!IsValidData)
        {
            OutputResult = "Невозможно сохранить результат: данные невалидны";
            return;
        }

        try
        {
            var file = await window.StorageProvider.SaveFilePickerAsync(new FilePickerSaveOptions
            {
                Title = "Сохранить результат",
                DefaultExtension = "txt",
                SuggestedFileName = $"intersection_result_{DateTime.Now:yyyyMMdd_HHmmss}.txt"
            });

            if (file is not null)
            {
                using var stream = await file.OpenWriteAsync();
                using var writer = new StreamWriter(stream);
                await writer.WriteAsync(OutputResult);
                await writer.FlushAsync();
                OutputResult += $"\n\n✓ Результат сохранен в файл: {file.Name}";
            }
```

#### B.6. Views/MainWindow.axaml.cs (основные части)

```csharp
using System;
using Avalonia;
using Avalonia.ReactiveUI;
using Avalonia.Controls;
using CircleIntersectionApp.ViewModels;
using Avalonia.Controls.Shapes;
using Avalonia.Media;
using ReactiveUI;
using System.ComponentModel;

namespace CircleIntersectionApp.Views;

public partial class MainWindow : ReactiveWindow<MainWindowViewModel>
{
    public MainWindow()
    {
        InitializeComponent();

        this.WhenActivated(_ =>
        {
            if (DataContext is MainWindowViewModel vm)
            {
                vm.PropertyChanged += ViewModel_PropertyChanged;
                DrawGraph(vm);
            }
        });
    }

    private void ViewModel_PropertyChanged(object? sender, PropertyChangedEventArgs e)
    {
        if (sender is MainWindowViewModel vm &&
            (e.PropertyName == nameof(MainWindowViewModel.IsValidData) ||
             e.PropertyName == nameof(MainWindowViewModel.OutputResult) ||
             e.PropertyName == nameof(MainWindowViewModel.CurrentCircleData)))
        {
            DrawGraph(vm);
        }
    }

    private void GraphCanvas_SizeChanged(object? sender, SizeChangedEventArgs e)
    {
        if (DataContext is MainWindowViewModel vm)
        {
            DrawGraph(vm);
        }
    }

    private void DrawGraph(MainWindowViewModel vm)
    {
        if (GraphCanvas is null)
            return;

        GraphCanvas.Children.Clear();

        if (!vm.IsValidData)
            return;

        var c = vm.CurrentCircleData;
        if (c.R1 <= 0 || c.R2 <= 0)
            return;

        double xMin = Math.Min(c.X1 - c.R1, c.X2 - c.R2);
        double xMax = Math.Max(c.X1 + c.R1, c.X2 + c.R2);
        double yMin = Math.Min(c.Y1 - c.R1, c.Y2 - c.R2);
        double yMax = Math.Max(c.Y1 + c.R1, c.Y2 + c.R2);

        double width = GraphCanvas.Bounds.Width;
        double height = GraphCanvas.Bounds.Height;
        if (width <= 0 || height <= 0)
            return;

        const double margin = 30;
        double dx = Math.Max(1.0, xMax - xMin);
        double dy = Math.Max(1.0, yMax - yMin);
        double scale = Math.Min((width - 2 * margin) / dx, (height - 2 * margin) / dy);
        double ConvX(double x) => margin + (x - xMin) * scale;
        double ConvY(double y) => height - margin - (y - yMin) * scale;

        var xAxis = new Line
        {
            StartPoint = new Point(0, ConvY(0)),
            EndPoint = new Point(width, ConvY(0)),
            Stroke = Brushes.LightGray,
            StrokeThickness = 1
        };
        var yAxis = new Line
        {
            StartPoint = new Point(ConvX(0), 0),
            EndPoint = new Point(ConvX(0), height),
            Stroke = Brushes.LightGray,
            StrokeThickness = 1
        };
        GraphCanvas.Children.Add(xAxis);
        GraphCanvas.Children.Add(yAxis);

        void DrawCircle(double cx, double cy, double r, IBrush strokeColor, double thickness, IBrush fillColor)
        {
            var ellipse = new Ellipse
            {
                Width = 2 * r * scale,
                Height = 2 * r * scale,
                Stroke = strokeColor,
                StrokeThickness = thickness,
                Fill = fillColor
            };
            Canvas.SetLeft(ellipse, ConvX(cx) - r * scale);
            Canvas.SetTop(ellipse, ConvY(cy) - r * scale);
            GraphCanvas.Children.Add(ellipse);

            var center = new Ellipse
            {
                Width = 6,
                Height = 6,
                Fill = Brushes.Black
            };
            Canvas.SetLeft(center, ConvX(cx) - 3);
            Canvas.SetTop(center, ConvY(cy) - 3);
            GraphCanvas.Children.Add(center);
        }

        DrawCircle(c.X1, c.Y1, c.R1, Brushes.DodgerBlue, 2, Brushes.Transparent);
        DrawCircle(c.X2, c.Y2, c.R2, Brushes.OrangeRed, 2, Brushes.Transparent);

        if (c.CirclesIntersect())
        {
            var ip = c.GetIntersectionPoints();
            if (ip.HasValue)
            {
                void DrawPoint(double x, double y)
                {
                    var p = new Ellipse
                    {
                        Width = 10,
                        Height = 10,
                        Fill = Brushes.Red
                    };
                    Canvas.SetLeft(p, ConvX(x) - 5);
                    Canvas.SetTop(p, ConvY(y) - 5);
                    GraphCanvas.Children.Add(p);
                }

                DrawPoint(ip.Value.x1, ip.Value.y1);
                DrawPoint(ip.Value.x2, ip.Value.y2);
            }
        }
    }
}
```
